What is the difference between TPC/UDP?
TCP is a stateful protocol and UDP is stateless.
TCP is for wireless communication and UDP for desktop communication
UDP is an encrypted protocol while TCP can be both.
In the local area connection, the computers communicate with each other in TCP while in the World Wide Web the computers communicate with UDP.
