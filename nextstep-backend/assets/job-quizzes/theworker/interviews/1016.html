2. איך למצוא את איבר הn לפני הסוף ברשימה מקושרת?
3.למצוא איזה מספר חסר במערך של 99 איברים שכל הערכים הם מ1-100 ואינם חוזרים על עצמם?
4. למצוא 2 מספרי מינימום במערך.
תשובות הוסף תשובה   |   לצפיה בתשובות
 
  פברואר 2020
  1. להגדיר שני מצביעים כשאר אחד מהם מתקדם בכל איטרציה צעד אחד והשני שני צעדים אם אי פעם הם יצביעו על אותו איבר זה אומר שיש מעגל, אם שניהם הגיעו לNULL אין מעגל.
2. להגדיר שני מצביעים לראש הרשימה, עם מצביעה אחד להתקדם n-1 צעדים קדימה, לאחר מכן להתקדם עם שני המצביעים צעד אחד בכל פעם עד שהמצביע הראשון יגיע לNULL, באותו הזמן המצביע השני יציע על האיבר שאנחנו מחפשים.
3. הסכום של כל האיברים מ1 עד 100 הוא 5050, עוברים על המערך ומחסרים את כל האיברים מסכום זה, ההפרש שנשאר הוא האיבר החסר.
4.מגדירים את ערכים כMAX_INT ומוודאים כי במערך אכן יש יותר מ2 איברים. לאחר מכן עוברים על כל המערך ומבצעים:
for (i = 0; i < arr_size ; i ++)
{
if (arr[i] < first)
{
second = first;
first = arr[i];
}
else if (arr[i] < second && arr[i] != first)
second = arr[i];
