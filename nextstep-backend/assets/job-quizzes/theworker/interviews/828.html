שאלות מתוך הראיון משימת תכנות: 1. נתונה מטריצה דו מימדית, יש להדפיס את האיברים בצורת ספירלה 2. במשחק שבו שני שחקנים, נתונים מספר מקלות וכל שחקן יכול לקחת בתורו 1, 2, או 3 מקלות, כתוב פונקציה שתחזה אם במשחק במצב מסוים שחקן יכול לנצח
במשימת הבית נדרשים להציג מצגת עם איפיון חנות אונליין עם מוצרים, מחירים, מבצעים, עגלת קניות ומסך סיכום. איפיון ברמת database and rest API
