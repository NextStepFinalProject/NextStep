שאלות מתוך הראיון Given an array containing only 0s and 1s, find the largest subarray which contains equal no of 0s and 1s. The expected time complexity is O(n).
Example:
Input: [0 , 1 , 0, 1 , 1 ]
Output: 4 (index 0 to 3)

sum = 0
maxi = 0
sum_map = {}
for i in range(len(arr)):
sum = (arr[i] == 1)? sum+1: sum-1
if sum == 0:
maxi = i+1
if sum in sum_map:
maxi = max(maxi, i-sum_map[sum])
else:
sum_map[sum] = i
return maxi
