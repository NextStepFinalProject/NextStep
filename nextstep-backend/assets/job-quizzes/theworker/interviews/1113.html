attr_accessor :first_on_top_queue, :last_on_top_queue

def initialize
@first_on_top_stack = []
@last_on_top_stack = []
end

def push(obj)
while !@first_on_top_queue.empty?
@last_on_top_queue.push(@first_on_top_queue.pop())
end
@last_on_top_queue.push(obj)
end

def pop
while !@last_on_top_queue.empty?
@first_on_top_queue.push(@last_on_top_queue.pop())
end
if @first_on_top_queue.empty?
puts "Nothing to pop()"
end
return @first_on_top_queue.pop()
end

end

ראיון לתפקיד פיתוח בחברת יורקסיס
מאי 2019
28.10.2019
 
פרטים לגבי התהליך ישנו ראיון אשר חלקו הראשון כולל הכרות עם המועמד, שאלות אשר מופנות לגבי דברים הכתובים בקורות חיים של המועמד, מתן הסברים על עבודות/ פרויקטים קודמים וכו'.

חלקו השני של הראיון כלל שתי שאלות מקצועיות, כאשר כל פעם מוצגת בעיה ויש לתת הסבר כיצד ניתן לפתור את הבעיה בעל פה בצורה מובנת או לכתוב קוד על דף.

לאחר מכן המראיינים מחליטים אם המועמד מתאים לצוות הכולל צד אלקטורני או לצוות הפיתוח.
אם מתאים לצוות הפיתוח - ישנה שאלה נוספת, אשר קשורה להתעסקות עצמה של החברה (בעיית זיהום האוויר בבניינים), נותנים סיטואציה מסויימת ושואלים מספר שאלות הגיוניות ועל המראוין לתת תשובות ולהסביר את עצמו.
אם לצוות השני - ניתנת בעיה אלקטרונית בשימוש עם מחשב.
שאלות מתוך הראיון השאלות המקצועיות שניתנו בראיון:

1. ישנו מערך אשר מייצג את שווי החברה בזמנים שונים, כאשר כל תא במערך מחזיק את שווי החברה לפי סדר כרונולגי (למשל: תא 0 מחזיק את שווי החברה ביום ה-30 של החברה, תא 1 מחזיק את שווי החברה ביום ה-100 של החברה, תא 2 מחזיק את שווי החברה ביום ה-102 של החברה וכן הלאה).
צריך להחזיר את האינדקס של התא במערך אשר מחזיק את שווי החברה האופטימלי שבו כדאי "למכור את החברה", (כלומר השווי הגבוה ביותר כדי להרוויח כמה שיותר מהחברה) בזמן הכי יעיל שאפשר (פחות מn^2).

2. בהינתן מערך של מספרים צריך להחזיר את זוג המספרים שסכומם הוא מקסימלי מבין כל הזוגות המספרים במערך.
יש לעשות זאת בזמן היעיל ביותר (פחות מ-n^2).

3. עבור השאלה הנוספת לצוות הפיתוח מציירים בניין על הלוח ושואלים שאלות כמו: מה צריך להיות בבניין כדי שתהיה זרימה של אוויר(פתחי איוורור), איך תתבצע יציאה וכניסה של אוויר מהבניין, עבור אחוזים מסויימים של זיהום אוויר איזה אחוזים של אוויר צריכים לבצע תחלופה עם אוויר נקי באיזה פרק זמן וכו' (שאלות הכוללות היגיון פשוט וחישובים פשוטים של אחוזים).
