פרטים לגבי התהליך שיחה ראשונה עם מנהל אגף ה DevOps ועם איש DevOps קיים.
גישה לסביבת AWS לצורך מטלת בית
שיחה שלישית לצורך מעבר על התרגיל
שיחה רביעית עם HR
המלצות.
סה"כ הראיונות עברו מעולה והתחושה שזו חברה עם אחלה הון אנושי אך לצערי, מסיבה שאינה ברורה החליטו לא להמשיך בתהליך
שאלות מתוך הראיון The assignment have 3 goals:
1.Create an EKS cluster in region us-east-1
With instance type t3.small
2.deploy the microservice (linked in the github account) to the EKS cluster and expose the service through a load balancer
3.create a CI/CD (jenkins , github action , aws codepipeline or any other tool you choose) for the deployment process
Bonus:
Create terraform / cloudformation templates for the 3 sections.
Please take bullet notes of your actions, and all access links.
