פרטים לגבי התהליך *המשרה כוללת כתיבת קוד בכמה שפות.
*משרה מלאה.
*מיקום החברה: נתניה (חייבים להגיע פיזית למשרד לפחות 3 ימים בשבוע).
שאלות מתוך הראיון שלב ראשון:
מבחן טכני:
- 3 שאלות, שתיים כתיבת קוד ושאלה אמריקאית.
- קודם יש דימו (לא נחשב בציון ולא חובה לעשות אותו), שאלת הדימו היא:
Given an array A of N integers, returns the smallest positive integer (greater than 0) that does not occur in A.
- משך הבחינה הוא 55 דק', משך הדימו 30 דק'.
שאלה 1 מהמבחן:
Write a function {Public int[] solution(int N);}
that, given an integer N (1<=N<=100), returns an array containing N unique integers that sum up to 0. The function can return any such array

שאלה 3:
Write a function solution that, given an integer N, returns the maximum possible value obtained by inserting '5' digit inside the decimal representation of integer N. (N is an integer within the range [-8000...8000])
